Operational Capabilities

Night Mode

Adaptive CLAHE enhancement and lowered detection thresholds for low-light environments.

IP Camera Support

Multi-Angle Onboarding

Captures Front, Left, Right, Back, and Top profiles for robust re-identification.

Stillness Detection

Engineering Protocols

Recommended
  • Use config.json for sensitivity tuning.
  • Ensure audio_files/ contains 'siren.mp3'.
  • Use RTSP with TCP transport for stability.
  • Calibrate Night Mode thresholds on site.
Risks & Warnings
  • High CPU usage with multiple RTSP streams.
  • Avoid special chars in Guard Names (OS limits).
  • Windows Console Unicode errors (Use SafeLogger).
